# MSME Business Assistant

AI-powered business assistant for MSME (Micro, Small, and Medium Enterprises) owners. Get insights from your business data and receive expert business advice through a conversational interface.

## Features

-  **Business Data Queries** - Query your sales, inventory, products, and expenses
- **Business Advice** - Get expert recommendations and best practices
- **Secure** - Read-only database access with business isolation
-  **Real-time** - WebSocket-based chat interface
-  **Private Network** - Accessible via Tailscale VPN
